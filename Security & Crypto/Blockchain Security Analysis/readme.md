# 🔒 Blockchain Security Analysis



This repository contains an HTML-based explanation of blockchain systems, an alternative approach to an original assignment, and a small experiment using GitHub Pages deployment.



The focus of this repository is not implementation complexity, but structural understanding of security systems and how they behave under adversarial conditions.
